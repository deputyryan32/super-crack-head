## Use as Extension

This repository can be added as an extension in MakeCode Arcade.

Steps:
- Open https://arcade.makecode.com/
- Click New Project
- Open the gear menu
- Select Extensions
- Search for:
  https://github.com/deputyryan32/super-crack-head
- Import the extension

## Edit This Project

To edit this repository in MakeCode:

- Open https://arcade.makecode.com/
- Click Import
- Select Import URL
- Paste:
  https://github.com/deputyryan32/super-crack-head
- Click Import
